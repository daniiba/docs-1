You can follow [this guide](https://docs.plasmo.com/quickstarts/with-mellowtel) to set up Mellowtel with an extension developed in the Plasmo framework

[Plasmo](https://www.plasmo.com/) is a framework that allows you to quickly build and deploy browser extensions that work on all major browsers.
